Generate Random Numbers using While Loop

Basic While Loop

Repeats code while a condition is true.

let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
// prints: 1 2 3 4 5

Math.random()

Generates a random number between 0 and 1.

let rand = Math.random();
console.log(rand);  // e.g., 0.7342

Random Integer in a Range

Formula: Math.floor(Math.random() * range) + min

// Random between 1 and 10
let num = Math.floor(Math.random() * 10) + 1;
console.log(num);  // e.g., 7

Generate Until a Target

Keep generating until we get a specific number.

let target = 5;
let attempts = 0;
let roll = 0;

while (roll !== target) {
    roll = Math.floor(Math.random() * 10) + 1;
    attempts++;
    console.log("Roll: " + roll);
}
console.log("Got 5 in " + attempts + " attempts");

Generate N Random Numbers

Use a counter with while loop.

let count = 1;
while (count <= 5) {
    let rand = Math.floor(Math.random() * 100) + 1;
    console.log("Random #" + count + ": " + rand);
    count++;
}

Do-While Loop

Runs at least once, then checks condition.

let num;
do {
    num = Math.floor(Math.random() * 6) + 1;
    console.log("Dice: " + num);
} while (num !== 6);
// keeps rolling until 6 appears
